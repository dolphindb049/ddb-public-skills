# DolphinDB Public Skills

A curated set of open-source Agent Skills for DolphinDB scripting, data engineering and operations.

## Skills

| Skill | Description |
|-------|-------------|
| [execute-dlang](./execute-dlang) | Python 持久化会话中分步执行 DolphinDB 脚本 |
| [ddb-data-analysis](./ddb-data-analysis) | 数据聚合、收益率建模、假设检验等标准分析工作流 |
| [ddb-data-discovery](./ddb-data-discovery) | 全库扫描、表结构透视、数据清洗与内存共享 |
| [ddb-cross-node-sync](./ddb-cross-node-sync) | 跨节点数据拉取与落地同步 (xdb + remoteRun) |
| [ddb-visualization](./ddb-visualization) | 参数化图表编排，生成金融业务报告 |
| [ddb-deployment-skill](./ddb-deployment-skill) | 单节点部署、高可用集群搭建与安全改造 |

## Usage

Each skill contains a `SKILL.md` with detailed instructions. Point your AI agent to the corresponding `SKILL.md` to activate the skill.
